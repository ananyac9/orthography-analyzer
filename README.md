# Orthography Analyzer

An NLP-based tool that analyzes the orthographies of texts, flags words with low grapheme-to-phoneme correlation, and suggests phonetic respellings and synonyms to make reading easier for individuals with orthographic dyslexia.

It currently supports English, Hindi, French, German and Finnish

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/ananyac9/orthography-analyzer 
   cd orthography-analyzer
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start the dev server**
   ```bash
   npm run dev
   ```
